Most people in digital careers follow a set path. Mine's always been about one thing running through it all.

I've done UX, web development, product, and marketing. Different jobs, different teams, different problems. But the same question keeps coming up: are we actually building this for the people who need to use it?

The answer is almost never a simple yes or no. That's the stuff I like to dig into here.
